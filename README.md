# Stock_Market_Forecasting
-->Description :

Stock_Market_Forecasting using plotly and dash. This project creates charts based on the ticker provided and forecasts the stock proces for the number of days provided in the imput using SVM(Suppot Vector Machines).

-->Dependencies :

- Plotly
- Dash
- Bootstrap
- pip

--> Setup

-  cd stockforecasting

-  pip install -r requirements.txt

-  python run.py
